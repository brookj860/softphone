📋 Twilio — where to find these
1. Go to console.twilio.com
2. Account SID & Auth Token — on the dashboard homepage
3. Phone Number — Phone Numbers → Manage → Buy a number
4. TwiML App SID — Voice → TwiML Apps → Create new
Set Voice URL to your app URL + /webhook/twilio/voice
5. API Key & Secret — Account → API Keys → Create
📋 Meta WhatsApp — where to find these
1. Go to developers.facebook.com
2. Create an App → Business type → Add WhatsApp product
3. Phone Number ID — WhatsApp → API Setup (15-digit number)
4. Permanent Token — Business Settings → System Users → Generate Token
Permissions: whatsapp_business_messaging
5. Verify Token — any string you choose. You'll paste it in Meta's webhook config too.
6. Webhook URL — WhatsApp → Configuration → set to your app URL + /webhook/whatsapp
✅
All set!
